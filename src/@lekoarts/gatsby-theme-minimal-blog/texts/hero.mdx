| 프로젝트                 | 부가설명                                                  | 모집 인원 |
| ------------------------ | --------------------------------------------------------- | --------- |
| NERA                     | 시립대 온라인 과제제출 플랫폼 기능추가 및 유지보수 개발자 | 00명      |
| Online Judge Site (가제) | 시립대 온라인 알고리즘 채점 플랫폼 개발자                 | 3명 ≤     |

| Kanban board             | 시립대 팀플 활동을 위한 칸반보드 플랫폼 개발자            | 2명 ≤     |
| 클라우드 기술팀           | CSUOS 개발자들을 위한 플랫폼 개발자                       | 1명 ≤     |
